## 1. Objectif du TP

* Gérer un compteur avec `useReducer`.
* Manipuler le focus d’un champ texte avec `useRef`.
* Compter le nombre de rendus d’un composant avec `useRef`.
* Créer un Hook personnalisé `useFetch` pour récupérer des données depuis une API.
* Créer un timer avec nettoyage automatique (`useEffect`).
* S’exercer avec des hooks personnalisés comme `useLocalStorage` et afficher la date/heure en temps réel.
